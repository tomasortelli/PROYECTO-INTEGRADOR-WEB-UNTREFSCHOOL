# Proyecto Integrador Desarrollo Web - Diplomatura UNTREF

El proyecto integrador contiene lo aprendido durante el primer tramo de la cursada Frontend.

Se realizó una página web de una inmobiliaria de Chivilcoy, mostrando sus propiedades, servicios y contacto. Las secciones planteadas se basan en las condiciones propuestas por el proyecto integrador, cambiando los contenidos acordes al rubro presentado.
